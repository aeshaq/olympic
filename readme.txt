How to run:
1. Create a virtual environment for flask and go into it
	a. install by using pip install virtaulenv
	b. run the command virtualenv env
	c. run the command env\Scripts\activate.bat
2. install the needed libraries (flask, pandas, plotly)
	a. run pip install flask pandas plot plotly-express
3. run the createDb.py file in the src to convert the csv to an sqlite database
4. run the command flask run
5. try out the different features!
